Hi, I'm Leonardo — an Android engineer with 5+ years of experience designing and shipping production mobile applications. I specialize in Kotlin, Jetpack Compose, Clean Architecture, and MVVM, with a focus on testability and code quality.

ProjectMiddleware started as a personal backend experiment to solve a real frustration: external APIs rarely return exactly what you need. Instead of writing bespoke transformation code in every app, I built a middleware layer that lets you define the transformation once and reuse it forever.
